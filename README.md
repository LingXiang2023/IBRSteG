# IBRSteG: Learning a Generalizable Steganography Framework for 3D Gaussian Splatting


> **Notice / 声明:** 
> The official code and datasets for the paper **"IBRSteG: Learning a Generalizable Steganography Framework for 3D Gaussian Splatting"** (IEEE Transactions on Multimedia) will be open-sourced in this repository. 
> 本仓库将用于开源 IEEE TMM 论文 **《IBRSteG: Learning a Generalizable Steganography Framework for 3D Gaussian Splatting》** 的官方代码和相关数据集。

## ⏳ Current Status / 当前状态

We are currently cleaning up the codebase for the **GAS (Gaussian Attributes Steganographer)** network and organizing the processing scripts for datasets (including DyNeRF, ENeRF, DTU, and THuman_MV) to ensure readability and usability. 
我们目前正在整理 **GAS (Gaussian Attributes Steganographer)** 网络的源代码，并完善相关数据集（DyNeRF, ENeRF, DTU, THuman_MV 等）的处理脚本，以确保代码的高可用性。

**Expected Release Date / 预计开源时间:** ** 2026年8月**
